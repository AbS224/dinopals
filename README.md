## About

DinoAlphabet Pals is an interactive, safe alphabet learning game for young children featuring a friendly dinosaur companion. The project runs entirely locally without requiring external services.

## Getting Started

The main project is located in the `DinoAlphabetPals-main` directory. Please see [DinoAlphabetPals-main/README.md](DinoAlphabetPals-main/README.md) for complete documentation, setup instructions, and features.

### Quick Start

```bash
cd DinoAlphabetPals-main
npm install
npm run dev
```

Visit http://localhost:3000 to start playing!
