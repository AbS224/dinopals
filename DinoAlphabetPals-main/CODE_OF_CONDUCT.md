# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

- Homepage: https://www.contributor-covenant.org
- Version: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- FAQ: https://www.contributor-covenant.org/faq

By participating, you agree to uphold a welcoming, inclusive, and respectful community. For reporting violations, contact the maintainers via GitHub issues or the private security advisory workflow if safety is a concern.

Attribution: This Code of Conduct is adapted from the Contributor Covenant, version 2.1, available at the link above.
